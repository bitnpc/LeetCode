## 204. Count Primes
### Difficulty

 <font color=green>Easy</font>

### Description

Count the number of prime numbers less than a non-negative number, **_n_**.

**Example:**
            Input: 10    Output: 4    ## Explanation: There are 4 prime numbers less than 10, they are 2, 3, 5, 7.
