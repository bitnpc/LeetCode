## 83. Remove Duplicates from Sorted List
### Difficulty

 <font color=green>Easy</font>

### Description

Given a sorted linked list, delete all duplicates such that each element
appear only _once_.

**Example 1:**
            Input: 1->1->2    Output: 1->2    

**Example 2:**
            Input: 1->1->2->3->3    Output: 1->2->3
