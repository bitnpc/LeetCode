## 22. Generate Parentheses
### Difficulty

 <font color=orange>Medium</font>

### Description

Given _n_ pairs of parentheses, write a function to generate all combinations
of well-formed parentheses.

For example, given _n_ = 3, a solution set is:
            [      "((()))",      "(()())",      "(())()",      "()(())",      "()()()"    ]
